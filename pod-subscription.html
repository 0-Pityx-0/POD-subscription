📛 ПРАВИЛА ПОДПИСКИ POD

1. Общие положения

1.1. Использование устройства только при активной подписке.
1.2. Без подписки устройство блокируется.

2. Виды подписок

По сроку: Недельная (7 дней), Месячная (30 дней).
Рабочая: дни сгорают даже без парения.
Свободная: тратятся только дни с затяжкой, остальные переносятся.

3. Продление

Остатки не суммируются. Переход между типами — только после истечения текущей.
